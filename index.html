Model Overview

Creation & Capture This model was created using the Scaniverse mobile application in splat mode to capture detailed 3D geometry.
Enhancement & Export After capturing, the model was enhanced with softer rendering and then exported as a .ply (Point Cloud) file and .html.
SuperSplat Processing The model was imported into SuperSplat Editor for optimization. Unnecessary geometric details were cleaned to improve quality and reduce file size.
Animation & Keyframes Strategic keypoints were added to create dynamic motion videos and interactive presentations.
Web Export Finally, the model was exported as HTML and .ply, optimized for web deployment without requiring additional plugins.
